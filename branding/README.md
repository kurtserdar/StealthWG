# StealthWG — Wraith Logo Kit

Marka rengi paleti:
- Gümüş (ana):  #D7DEE9
- Teal (bağlı durum aksanı): #34E0C4
- Mürekkep (açık zemin): #1A2130
- Zemin koyu: #0B0E14 / #12161F
- Metin: #F2F5F8, ikincil #8A94A6

Fontlar: Space Grotesk (wordmark), JetBrains Mono (etiketler) — Google Fonts.
Lockup SVG'leri font yüklü sistemde doğru görünür; baskı/sunum için fontu gömmek veya
metni eğriye çevirmek (outline) gerekebilir.

## Dosyalar
- mark-silver.svg — ana mark, şeffaf zemin (koyu arka planlar için)
- mark-teal.svg — bağlı/aktif durum varyantı
- mark-ink.svg — açık zeminler için
- mark-mono-outline.svg — tek renk kontur (tray, damga, gravür)
- mark-solid.svg — küçük boyut / favicon (16-32px'te en net bu)
- icon-ios.svg — 27/120 köşe yarıçaplı squircle (1024px'e ölçekleyin)
- icon-android.svg — dairesel adaptive icon ön planı
- icon-macos.svg — kenar boşluklu macOS stili
- icon-windows.svg — keskin köşeli tile
- icon-linux-tray.svg — mono kontur, şeffaf
- lockup-horizontal.svg / lockup-vertical.svg — yazılı kilitli düzenler
